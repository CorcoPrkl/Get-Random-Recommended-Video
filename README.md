# Get-Random-Recommended-Video
Get's 100 random recommended video's on youtube, when given a video ID to start from.


usage:

node getNextVideo.js [VIDEO_ID] [TIMES_TO_RUN]

Will default to a certain video if no ID is given. Will default to 100 iterations if not specified.

Requires a youtube API browser key. https://console.developers.google.com/apis/library
